Overview

Anybeam was created with the goal to make the Transmission as easy and simple as Possible. But still there can be some problems with using our application. That's why we made this little page to help you with eventual starting problems.

If you still cant get our application to work, hav a good suggestion for an enhancement or just want to say how much you love Anybeam, you can email us. You find the Address on our about site.

Devices & Systems

  • Android 4+
  • Windows 7,8 or 8.1
  • Mac OSx
  • Ubuntu 12.04+
  • Java 7+

If you want Anybeam on your android device simply download it from the Playstore.

For an desktop environment you need to have alt least Java 7 installed!

For an advanced installation guide see the Installation chapter.

Connecting

First make sure that your group settings (Group Password, Brodcast Port and Encryption Type) are equal on all your devices.

And Anybeam running on each Device you want to send or recieve data.

Android

On android it is very simple, just use the share function you can find in many other applications. You will see that you can share most of your data.

After you have opened Anybeam on your Android device, select your target from the list of devices with running Anybeam.

Desktop

For your desktop environment start Anybeam and select what kind of data you want to send.
